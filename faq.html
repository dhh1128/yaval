    How do I install yaval?

    The yaval tool is a python script. It doesn't require any fancy installation; just get the code and run it. See the Tool section of the overview.

    to have PyYAML installed (pip install PyYaml). Yaval schemas are yaml 1.1 documents. The yaval tool uses python's PyYAML library, which does not yet support yaml 1.2. In practice, this distinction is mostly unimportant.

    The documents that yaval validates must be parseable by a yaml 1.1 parser as well, which means that old yaml may not work, and corner-case yaml 1.2 might also cause hiccups. If we complete the task of adding a C++ impl of yaval based on yaml-cpp, we will have full yaml 1.2 support.

    Why re-invent the wheel instead of just enhancing json-schema or kwalify?

    The less interesting answer is that all the schema alternatives we looked at felt like they were either too clunky or too verbose, and I couldn't find a python API for validation.

    The deeper answer is the lacuna humana problem that no schema language or serialization spec has solved. Until now...
